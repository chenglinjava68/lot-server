# LotteryServer

#### 介绍
springboot整合shiro的权限管理系统（前后端分离）

#### 软件架构
软件架构说明
|  分类   |  技术栈   |   备注    |
| --- | --- | --- |
|  JDK  |  java1.8   |     |
|   编译  |   Maven  |     |
|   核心  |   Spring Boot Spring Spring MVC  |     |
|  JDBC   |  tomcat-jdbc  mybatis JPA   |     |
|  日志   |  slf4j、logback(皆暂无)   |     |
|  应用服务器   |   Tomcat  |     |
|   API  |  Swagger   |     |
|   报表 |  jasperreport   |     |
|   源码 |  git、gitee   |     |
|   前端 |  React+dva+ant Design pro   |     |
|   数据库 |  MYSQL5.7   |     |

#### 使用说明

1.  使用idea克隆源码库
2.  启动项目
